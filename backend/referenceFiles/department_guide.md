# 就诊科室指引

> 不确定挂哪个科？本指南帮你快速定位，减少跑冤枉路。

---

## 快速索引：按症状找科室

| 症状 | 推荐科室 |
|------|---------|
| 发热（不明原因）| 内科 / 发热门诊 |
| 发热 + 咳嗽 + 咽痛 | 呼吸内科 / 内科 |
| 胸痛、心慌、气促 | 心内科 / 急诊 |
| 腹痛、腹泻、消化不良 | 消化内科 / 普通外科（急性腹痛）|
| 头痛、头晕、记忆力减退 | 神经内科 |
| 手脚麻木、肢体无力 | 神经内科 |
| 关节痛、肌肉痛 | 风湿免疫科 / 骨科 |
| 腰痛、颈椎不适 | 骨科 / 脊柱外科 |
| 外伤、骨折 | 骨科 / 急诊外科 |
| 皮疹、瘙痒 | 皮肤科 |
| 视力下降、眼红眼痛 | 眼科 |
| 耳鸣、听力下降、耳痛 | 耳鼻喉科 |
| 鼻塞、流涕、鼻出血 | 耳鼻喉科 |
| 咽喉痛（严重）| 耳鼻喉科 |
| 牙痛 | 口腔科 |
| 尿频、尿急、尿痛 | 泌尿外科 / 内科（女性可先妇科）|
| 血尿 | 泌尿外科 |
| 月经异常、白带异常 | 妇科 |
| 怀孕检查、产检 | 产科 |
| 儿童发热（<14岁）| 儿科 |
| 心情低落、失眠、焦虑 | 心理科 / 精神科 |
| 血糖异常、甲状腺问题 | 内分泌科 |
| 血压异常 | 心内科 / 内科 |
| 贫血、白细胞异常 | 血液科 / 内科 |
| 肿块（不明原因）| 普通外科 / 相关专科 |

## 各科室详细介绍

### 内科（门诊常见首选）
**擅长**：不明原因发热、全身性疾病初步诊断、高血压、糖尿病常规管理  
**适合人群**：症状不明确、不知道挂哪科时，内科是好的起点

### 急诊科
**适用情况**（以下情况请直接去急诊，勿预约排队）：
- 持续胸痛/呼吸困难
- 突发意识丧失或抽搐
- 严重外伤、大量出血
- 急性腹痛（疑似阑尾炎、肠梗阻）
- 高热伴精神状态改变
- 疑似脑卒中（口角歪斜、肢体无力、言语不清）

---

### 心内科（心脏内科）
**擅长**：冠心病、心肌梗死、心衰、心律不齐、高血压  
**常用检查**：心电图、心脏彩超、冠状动脉造影

### 呼吸内科
**擅长**：肺炎、哮喘、慢阻肺、肺结核、不明原因咳嗽  
**常用检查**：肺功能检查、胸部 CT、痰培养

### 消化内科
**擅长**：胃炎、胃溃疡、肠炎、肝炎、炎症性肠病  
**常用检查**：胃镜、肠镜、腹部 B 超

### 神经内科
**擅长**：偏头痛、脑卒中、癫痫、帕金森病、痴呆、周围神经病  
**常用检查**：头颅 CT/MRI、脑电图、肌电图

### 内分泌科
**擅长**：糖尿病、甲亢、甲减、肾上腺疾病、骨质疏松  
**常用检查**：血糖、甲状腺功能、激素水平

### 风湿免疫科
**擅长**：类风湿关节炎、系统性红斑狼疮、强直性脊柱炎、痛风  
**常用检查**：类风湿因子、ANA、血尿酸

### 血液科
**擅长**：贫血、白血病、淋巴瘤、血小板异常  
**常用检查**：血常规、骨髓穿刺

---

### 骨科
**擅长**：骨折、关节置换、脊柱疾病、运动损伤  
**分科**：脊柱外科、关节外科、运动医学科

### 普通外科
**擅长**：阑尾炎、胆囊炎、疝气、甲状腺手术、乳腺外科  
**常用检查**：腹部 B 超、腹部 CT

### 泌尿外科
**擅长**：尿路感染、肾结石、前列腺疾病、膀胱肿瘤

### 神经外科
**擅长**：脑肿瘤、颅内出血、脑积水、椎间盘手术

---

### 妇科
**擅长**：月经病、子宫肌瘤、卵巢囊肿、宫颈检查、炎症  
**注意**：未成年或首次就诊建议选择女性医生

### 产科
**擅长**：孕期检查、高危妊娠、分娩  
**建议**：确认怀孕后尽早建档（通常孕 6–8 周）

### 乳腺科（部分医院属普外科）
**擅长**：乳腺结节、乳腺炎、乳腺癌筛查  
**建议**：20 岁以上女性每年乳腺 B 超检查

---

### 儿科
**适用**：14 岁以下儿童（部分医院 18 岁）  
**注意**：儿童用药剂量与成人不同，切勿照搬成人用药

### 皮肤科
**擅长**：湿疹、痤疮、荨麻疹、银屑病、皮肤感染、脱发

### 眼科
**擅长**：近视、白内障、青光眼、眼底病变、眼外伤  
**建议**：40 岁后定期检查眼压（排查青光眼）

### 耳鼻喉科（ENT）
**擅长**：中耳炎、鼻窦炎、扁桃体炎、鼻息肉、耳鸣

### 口腔科
**擅长**：龋齿、牙周病、口腔溃疡、智齿、正畸  
**建议**：每 6 个月进行一次口腔检查和洁牙

### 心理科 / 精神科
**擅长**：焦虑症、抑郁症、失眠障碍、双相障碍  
**说明**：心理科多侧重咨询，精神科可开具处方药；两者常可转介

---

## 常见检查说明

| 检查项目 | 主要用途 |
|---------|---------|
| 血常规 | 炎症、贫血、血液病初筛 |
| 尿常规 | 泌尿系统感染、肾功能初筛 |
| 血生化 | 肝功能、肾功能、血脂、血糖 |
| 心电图 | 心律异常、心肌缺血 |
| B 超 | 腹腔脏器、血管、甲状腺、乳腺 |
| X 线 | 骨折、肺部粗筛 |
| CT | 胸腹部病变、脑部 |
| MRI | 脑、脊髓、软组织（无辐射）|
| 胃镜/肠镜 | 消化道病变（溃疡、息肉、肿瘤）|

## 就诊小贴士

1. **提前整理症状**：发病时间、性质、持续时间、加重/缓解因素
2. **带好既往资料**：历史检查报告、用药记录、过敏史
3. **空腹要求**：抽血（血糖、血脂、肝功能）、腹部 B 超需空腹 8 小时
4. **陪同就诊**：老人、儿童、精神状态不佳者建议有人陪同
5. **互联网医院**：轻症随访、慢病复诊可考虑线上问诊，节省时间
