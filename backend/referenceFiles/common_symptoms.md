# 常见症状与疾病对照手册

> 本手册供 AI 医疗问答助手参考使用，不能替代专业医生诊断。

---

## 一、发热（体温升高）

### 症状描述
- 低热：37.3°C–38°C
- 中等热：38.1°C–39°C
- 高热：39.1°C–41°C
- 超高热：>41°C

### 常见原因
| 伴随症状 | 可能疾病 |
|---------|---------|
| 咳嗽、流涕、咽痛 | 上呼吸道感染（普通感冒）、流行性感冒 |
| 咳嗽、胸痛、呼吸急促 | 肺炎、支气管炎 |
| 尿频、尿急、尿痛 | 泌尿道感染 |
| 腹痛、腹泻、恶心 | 急性胃肠炎、食物中毒 |
| 皮疹、关节痛 | 病毒感染、风湿性疾病 |
| 持续高热无明显局部症状 | 需排除败血症、结核等，建议就医 |

### 处理建议
- 低热可多休息、多喝水，监测体温变化
- 体温超过 38.5°C 可适当服用退烧药（如布洛芬、对乙酰氨基酚）
- 高热持续超过 3 天、儿童高热伴惊厥、老人高热，**立即就医**

## 二、头痛

### 常见类型与特征
| 类型 | 特征 | 可能原因 |
|------|------|---------|
| 紧张性头痛 | 双侧压迫感，钝痛，无搏动 | 压力、疲劳、颈椎问题 |
| 偏头痛 | 单侧搏动性疼痛，伴恶心/畏光 | 偏头痛病 |
| 丛集性头痛 | 眼眶周围剧烈疼痛，反复发作 | 神经血管异常 |
| 继发性头痛 | 伴发热、颈项强直、视力变化 | 脑膜炎、颅内高压（紧急！）|

### 处理建议
- 轻度头痛：休息、减压、适量饮水，可服用止痛药
- 突发剧烈头痛（"一生中最严重的头痛"）：**立即拨打急救电话**，可能是蛛网膜下腔出血
- 头痛伴发热、颈项强直：**紧急就医**

## 三、胸痛

> ⚠️ 胸痛需优先排除心脏急症，请务必重视。

### 鉴别要点
| 特征 | 可能原因 |
|------|---------|
| 压迫感、放射至左臂/下颌、出汗 | 急性心肌梗死（紧急！）|
| 活动后加重，休息缓解 | 心绞痛 |
| 锐痛，深呼吸/咳嗽加重 | 胸膜炎、肺栓塞 |
| 烧灼感，进食后明显 | 胃食管反流 |
| 局部压痛 | 肋软骨炎、肌肉拉伤 |

### 处理建议
- 疑似心脏原因的胸痛：**立即拨打 120**，停止活动，就地休息
- 有冠心病病史者，舌下含服硝酸甘油并呼叫急救

## 四、腹痛

### 按部位分类
| 部位 | 常见原因 |
|------|---------|
| 右上腹 | 胆囊炎、胆结石、肝炎 |
| 左上腹 | 胃炎、胃溃疡、脾脏问题 |
| 右下腹 | 阑尾炎（急性者需手术）、肠痉挛 |
| 左下腹 | 结肠炎、憩室炎 |
| 脐周 | 肠炎、肠梗阻早期 |
| 全腹剧痛、板状腹 | 消化道穿孔（紧急！）|

### 处理建议
- 轻度腹痛伴腹泻：补液、休息，可服用整肠生等益生菌
- 腹痛持续加重、伴发热或呕血：**立即就医**
- 疑似阑尾炎：**不要服用止痛药掩盖症状**，尽快就医

## 五、咳嗽

### 按时长分类
- **急性咳嗽**（<3 周）：多为感冒、急性支气管炎
- **亚急性咳嗽**（3–8 周）：感染后咳嗽、过敏
- **慢性咳嗽**（>8 周）：咳嗽变异性哮喘、鼻后滴漏、胃食管反流

### 伴随症状鉴别
| 伴随症状 | 可能原因 |
|---------|---------|
| 咳嗽 + 黄脓痰 + 发热 | 细菌性肺炎、支气管炎 |
| 干咳 + 夜间明显 | 咳嗽变异性哮喘 |
| 咳嗽 + 咳血 | 结核、肺癌（需排查）|
| 咳嗽 + 呼吸困难 | 哮喘急性发作、心衰 |

## 六、呼吸困难

### 常见原因
- **肺源性**：哮喘、慢阻肺、肺炎、气胸
- **心源性**：心衰、心肌梗死
- **其他**：贫血、焦虑发作、过敏反应（过敏性休克）

### 处理建议
- 突发呼吸困难伴喘鸣：哮喘可使用支气管扩张剂；无缓解立即就医
- 安静时呼吸困难、口唇发绀：**立即拨打 120**

---

## 七、恶心与呕吐

| 伴随症状 | 可能原因 |
|---------|---------|
| 腹泻 + 发热 | 急性胃肠炎、食物中毒 |
| 剧烈头痛 | 颅内压增高（紧急）|
| 黄疸 + 右上腹痛 | 肝胆疾病 |
| 妊娠期 | 早孕反应 |
| 服药后 | 药物不良反应 |

### 处理建议
- 少量多次补充电解质液，避免脱水
- 呕吐超过 24 小时或呕血：**就医**

## 八、皮疹

| 类型 | 特征 | 可能原因 |
|------|------|---------|
| 荨麻疹 | 风团样，瘙痒，消退不留痕 | 过敏 |
| 水泡疹 | 成簇，沿神经分布 | 带状疱疹 |
| 全身红斑 + 发热 | 病毒性皮疹（麻疹、风疹等）|
| 蝴蝶形红斑 | 面颊对称 | 系统性红斑狼疮 |
| 银白色鳞屑 | 境界清楚 | 银屑病（牛皮癣）|

---

## 九、关节痛

| 受累关节 | 特征 | 可能原因 |
|---------|------|---------|
| 单关节红肿热痛 | 急性发作 | 痛风、感染性关节炎 |
| 多关节对称性晨僵 | >1 小时 | 类风湿关节炎 |
| 膝关节磨损感 | 老年人 | 骨关节炎 |
| 游走性关节痛 + 发热 | 青少年 | 风湿热 |

## 十、血糖异常（糖尿病相关）

### 低血糖（血糖 < 3.9 mmol/L）
**症状**：心慌、出汗、手抖、头晕、饥饿感
**处理**：立即进食含糖食物（15g 糖，如 3–4 块方糖），15 分钟后复测；意识不清立即就医

### 高血糖
**症状**：多饮、多尿、多食、体重减轻
**处理**：按医嘱调整用药，监测血糖；若出现酮症酸中毒症状（恶心、呕吐、腹痛、呼吸有果味）**立即就医**
